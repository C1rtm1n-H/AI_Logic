### 3. 设 S 是子句集合，用 R(S) 表示 S 的消解闭包，即：如果 c ∈ S，则 c ∈ R(S)；如果 c1, c2 ∈ R(S)，且 c 是 c1 和 c2 的消解，则 c ∈ R(S)。当 S 为如下的子句集合时，求出 R(S)：
• {[p, ¬q], [p, q], [¬p]}
• {[p], [q], [p, q]}

1. 初始集合：C1：[p, ¬q]，C2：[p, q]，C3：[¬p]
    第一轮消解：
    - C1和C2：[p]
    - C1和C3：[¬q]
    - C2和C3：[q]
    第二轮消解：
    - [p]和C3：[]
    - [¬q]和[q]：[]
    最终结果：{[p, ¬q], [p, q], [¬p], [p], [¬q], [q], []}

2. 初始集合：C1：[p]，C2：[q]，C3：[p, q]
    由于子句中不存在反置文字，无法进行消解。
    最终结果：{[p], [q], [p, q]}
